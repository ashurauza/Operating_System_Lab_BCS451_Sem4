## 🚀 How to Use

1. **Browse PDFs:**  
   Download/view any experiment PDF from the repo.

2. **Run C Code:**  
   Copy C code from PDFs, compile with GCC:
   ```bash
   gcc experiment.c -o experiment
   ./experiment
   ```

3. **Learn & Modify:**  
   Tweak, run, and experiment to deepen your OS concepts!
